# SparkSample
The samples are based on Apache Spark by Maven or Gradle.
<p>
Requirements:       <br>
              Eclipse      <br>
              Gradle 2.x   <br>
              Maven 2.x    <br>
<p>
Gradle Project :  <br>
                gradle build <br>
<p>
Maven Project :  <br>
                mvn package  <br>
